github.com/JaimeP-0/T2U2-DWI

Objetivo del proyecto

Crear una landing page con HTML y CSS para un catálogo ficticio de películas y series, demostrando además el uso profesional de Git y GitHub con ramas, pull requests, revisión de código y resolución de conflictos.

Integrantes y roles

Integrante únicoJaime Puente
Proyecto individual
LíderCoordinación y decisiones
DocumentadorREADME y presentación
Integrador / DiseñadorPRs, merge y estilos visuales

Estructura del sitio web

  • Varias paginas HTML enlazadas (sin anclas #)
  • Header con navegacion entre paginas
  • Tarjetas de catálogo (películas, series, top semanal)
  • Presentación del equipo T2U2
  • Formulario para sugerir títulos

Flujo de trabajo Git

main ---- codigo estable (produccion)
|
+-- dev ---- HTML + CSS base
|
+-- diseno -- mejoras visuales
|
+-- PR -> revision -> merge -> main

Pull Request y revisión

  • PR creado desde dev hacia main
  • Revisión de código en rol de integrador (proyecto individual)
  • Comentarios sobre estructura HTML y accesibilidad
  • Aprobación antes del merge

Resolución de conflictos

  • Conflicto en index.html — etiqueta <title>
  • Rama dev: "CineSpot | Estrenos y Series"
  • Rama diseno: "CineSpot | T2U2 DWI"
  • Resolución: se mantuvo "CineSpot | T2U2 DWI"

Conclusión

El proyecto cumple con los requisitos de ramas, commits descriptivos, PR con revisión y resolución de conflictos. CineSpot es responsive y tiene estética de cine.
