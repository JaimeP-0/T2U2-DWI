Objetivo del proyecto

Crear una landing page con HTML y CSS para un sitio ficticio de apuestas de futbol, demostrando ademas el uso profesional de Git y GitHub con ramas, pull requests, revision de codigo y resolucion de conflictos.

Integrantes y roles

Integrante unicoJaime Puente
Proyecto individual
LiderCoordinacion y decisiones
DocumentadorREADME y presentacion
Integrador / DisenadorPRs, merge y estilos visuales

Estructura del sitio web

  • Varias paginas HTML enlazadas (sin anclas #)
  • Header con navegacion entre paginas
  • Tarjetas de partidos con cuotas de ejemplo
  • Presentacion del equipo T2U2
  • Formulario para registrar apuestas

Flujo de trabajo Git

main ---- codigo estable (produccion)
|
+-- dev ---- HTML + CSS base
|
+-- diseno -- mejoras visuales
|
+-- PR -> revision -> merge -> main

Pull Request y revision

  • PR creado desde dev hacia main
  • Revision de codigo en rol de integrador (proyecto individual)
  • Comentarios sobre estructura HTML y accesibilidad
  • Aprobacion antes del merge

Resolucion de conflictos

  • Conflicto en index.html — etiqueta <title>
  • Rama dev: "GolApuestas - Apuestas deportivas"
  • Rama diseno: "GolApuestas | T2U2 DWI"
  • Resolucion: se mantuvo "GolApuestas | T2U2 DWI"

Conclusion

El proyecto cumple con los requisitos de ramas, commits descriptivos, PR con revision y resolucion de conflictos. GolApuestas es responsive y tiene estetica de cancha de futbol.
